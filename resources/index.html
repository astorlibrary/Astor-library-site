Astor Study

Free literature resources.

A growing library of free study resources for classic texts, poetry, Shakespeare, drama, Victorian literature and modern fiction. Built for students, teachers and independent readers.

How to use this page. Start with a text or category, then open the individual resource page. Each page gives a short explanation, a preview image, a PDF download and related Astor links.

Not a resource dump.

Resources are grouped by text and category so this page can grow without becoming a long unstructured list.
